## API

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
|--- |--- |--- |--- |--- |
| status | 指定当前loading效果的状态, 0表示初始状态或已加载成功，不需要loading；1表示当前正在加载中；-1表示当前已加载失败。 | number | — | — |
| loaderHeight | 设置loader的默认高度 | number | — | 300 |
| LoadingTime | loading时间 | number | — | 300 |
| noThead | tableLoader指令设置loading时不显示thead | boolean | — | false |
